# 단어(Vocabulary) 이미지 → `words.json` 프롬프트

모바일 Claude 대화에 **단어장(한 DAY 분량) 사진**을 첨부하고, 아래 `---` 사이 내용을 그대로 붙여넣으세요.
받은 JSON을 리포 루트에 **`words.json`** 으로 저장한 뒤:

```
python scripts/import_words.py words.json
```

→ `data/vocabulary/dayNN.json` 저장 + `data/vocabulary/index.json` 재생성(최신 DAY가 위). 커밋·푸시하면 앱에 자동 반영됩니다.

---

첨부한 사진은 영어 단어장(한 DAY 분량)입니다. 표는 보통 **어휘(영어) · 뜻(한글)** 열로 되어 있습니다.
사진의 모든 단어를 **빠짐없이·사진 순서대로** 읽어 아래 JSON으로 만들어 주세요.

**뜻(한글) 표기 규칙 — 사진 원자료를 그대로 보존하되 대괄호만 제거:**
- 품사 표기 `[명]`·`[동]` 등은 **대괄호를 떼고 「글자+공백」** 으로: `[명] 와인잔` → `명 와인잔`.
  품사 글자 11종: **명 동 형 부 전 접 대 감 조 관 수**.
- 한 단어에 **품사가 둘 이상**이면 세미콜론 `;` 로 구분: `[동] ~을 탓하다 [명] 탓` → `동 ~을 탓하다; 명 탓`.
- 품사 표기가 **없는** 구·숙어는 그대로: `유리그릇`, `한동안`, `~을 무서워하다`.
- 한 품사 안에서 번호로 나뉜 여러 뜻(`1. … 2. …`)은 **한 줄로 그대로** (세미콜론 넣지 말 것).
- 쉼표·괄호·물결(~)·가운뎃점(·) 등 사진의 문장부호를 그대로 유지.

**정확도**: 철자·품사·뜻을 사진과 한 번 더 대조해 검증하세요. 흐릿해 확신이 안 서는 항목은 `⚠️` 로 표시해 되물어 주세요(추측 금지).

**출력 형식** (DAY 번호·제목은 사진 상단에서 읽어 `title` 에):

```json
{
  "groups": [
    {
      "title": "DAY 13  School Life",
      "words": [
        ["fashion", "명 유행, 패션"],
        ["design", "동 디자인하다, 설계하다; 명 디자인, 설계"]
      ]
    }
  ]
}
```

먼저 **검증표**(# · 영어 · 뜻)를 보여주고, 그 아래에 **위 JSON만 코드블록으로** 출력하세요.
